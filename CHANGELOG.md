# История изменений

Все заметные изменения в этой коллекции навыков отражаются в этом файле.

Формат основан на `Keep a Changelog`, а версии следуют `SemVer`.

## [Не выпущено]

## [0.2.0] - 2026-05-19

### Добавлено

- Добавлен корневой манифест `APM` в `apm.yml`.
- Добавлен `README.md` с инструкциями по установке через `APM`.
- Добавлен `LICENSE` с лицензией `MIT`.
- Добавлен пилотный сценарий проверки `APM` в `docs/apm-pilot.md`.
- Добавлен навык `project-readme` для сопровождения русскоязычного
  `README.md` проекта.
- Добавлен навык `primary-data-pipeline` для оркестрации конвейера первичных
  данных.
- Добавлен навык `primary-data-summary` для сводки состояния первичных данных.
- Добавлен навык `source-fact-extraction` для извлечения проверяемых фактов из
  первичных данных.

### Изменено

- Репозиторий переведён в формат коллекции навыков `APM`.
- Навыки перенесены в структуру `skills/<name>/`.
- Обновлены внутренние правила и база знаний под новую структуру.
- Расширены `source-inventory` и `source-impact-audit`: добавлены переносимые
  контракты слоя первичных данных, проверки индексов и правила работы с
  извлечёнными фактами.
